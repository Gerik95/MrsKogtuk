## README `Mrs.Kogtuk`

# Mrs.Kogtuk

Mrs.Kogtuk is a business card website created for a nail technician.

The project focuses on clean visual presentation, clear service information, and a user-friendly interface for potential clients.

## Overview

This project was built as a frontend website for a personal beauty service brand.  
The goal was to create a simple, responsive, and visually clear website that presents services in an understandable and professional way.

## Features

- Business card website structure
- Clear presentation of services
- Responsive layout
- Clean and simple UI
- Easy-to-read content structure
- Personal brand presentation

## Tech Stack

- React
- JavaScript
- HTML5
- CSS3

## What I Practiced

- Building a frontend website for a personal service brand
- Creating responsive layouts
- Structuring content for clarity and usability
- Improving visual presentation
- Designing a simple and user-friendly interface

## Project Goal

The goal of this project was to strengthen practical frontend skills by creating a website for a real-life service niche, with attention to responsive design, clean layout, and clear presentation of services.
